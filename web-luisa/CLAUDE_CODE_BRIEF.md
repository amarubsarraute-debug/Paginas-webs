# Brief para Claude Code

Objetivo: redisenar la landing de Dra. Luisa Cedeno desde el codigo fuente, no desde una captura.

Stack:
- Vite + TanStack Start + React + Tailwind CSS 4.
- Componentes principales en `src/routes/index.tsx`.
- Estilos globales en `src/styles.css`.
- Datos de tratamientos en `src/data/treatments.ts`.
- WhatsApp en `src/lib/whatsapp.ts`.

Comandos:
```bash
npm install
npm run dev
```
